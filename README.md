Dashboard no Power BI

Projeto desenvolvido na formação de Analista de Dados da EBAC.

Objetivo

Construir um dashboard completo no Power BI, passando pelas etapas de preparação dos dados, modelagem, criação de métricas em DAX e geração de insights para apoio à tomada de decisão.

Etapas realizadas

* Importação da base de dados
* Tratamento dos dados no Power Query
* Limpeza e tipagem das colunas
* Modelagem em esquema estrela
* Criação de medidas em DAX
* Desenvolvimento de KPIs
* Construção de tabelas e gráficos interativos
* Criação de segmentações e filtros
* Elaboração de insights sobre o negócio

Recursos utilizados

* Power Query
* DAX
* Modelagem de dados
* Dashboard interativo
* KPIs e indicadores de desempenho

Arquivos
